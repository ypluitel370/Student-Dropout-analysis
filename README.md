This project focuses on analyzing student dropout data to identify the key factors that influence students leaving school before completion. Using data analysis techniques in Python, the study examines variables such as attendance, academic performance, socio-economic background, and school environment. The goal is to uncover patterns and trends that can help educators and policymakers take early action to reduce dropout rates and improve student retention. The project includes data cleaning, visualization, and basic predictive analysis to support better decision-making in the education sector.
